Node.js Niveau I — Syllabus

Initiation Node.js (dernière LTS)

Durée : 3 jours
Format : présentiel ou distanciel synchrone
Référence version : branche LTS Node.js 24 (Krypton), état constaté au 30/04/2026

Description

Formation de 3 jours pour apprendre Node.js pas à pas, avec une progression orientée pratique. Les participants découvrent les fondamentaux (runtime, modules, asynchrone), puis construisent une API web complète avec Express, et terminent sur un atelier de synthèse en binôme.

Le parcours est gamifié pour garder un rythme dynamique : défis, points, niveaux et objectifs de sprint. Chaque notion technique est appliquée immédiatement dans un mini-univers de jeu.

Introduction historique (repères)

  • 2009 : création de Node.js par Ryan Dahl pour exécuter JavaScript côté serveur avec un modèle non bloquant.
  • 2010 : arrivée de npm, qui accélère fortement l'adoption grâce à l'écosystème de packages.
  • 2015 : stabilisation du projet sous la Node.js Foundation, puis convergence de l'écosystème.
  • Depuis : cadence LTS régulière, amélioration des performances, sécurité et expérience développeur.

Briques techniques qui constituent Node.js

  • Moteur V8 : exécution et optimisation du code JavaScript.
  • libuv : boucle d'événements, I/O non bloquantes, pool de threads.
  • APIs Node.js : modules natifs (fs, http, path, stream, etc.).
  • Système de modules : CommonJS et ESM pour structurer les applications.
  • npm : gestionnaire de packages, scripts de projet et dépendances.
  • Couche C/C++ interne : liaison entre JavaScript, V8 et les composants système.

Public cible

  • Profils souhaitant démarrer en programmation backend
  • Développeurs front souhaitant passer full stack
  • Professionnels en reconversion vers le développement web
  • Étudiants souhaitant comprendre les bases d'une API moderne

Prérequis

  • Aisance informatique de base (fichiers, terminal, éditeur)
  • Notions JavaScript recommandées (variables, fonctions, objets)
  • Aucune expérience Node.js requise

Objectifs pédagogiques

  1. Installer et configurer Node.js LTS, npm et un environnement de dev
  2. Comprendre le modèle Node.js : event loop, non bloquant, modules
  3. Écrire du code asynchrone avec Promises et async/await
  4. Construire une API REST avec Express (CRUD complet)
  5. Structurer un projet : routes, controllers, services, middleware
  6. Ajouter une base de données SQLite avec accès propre
  7. Tester et sécuriser une API
  8. Livrer un mini-projet en binôme à partir d'une expression des besoins

Fil rouge gamifié (J1 + J2) : Campus Quest API

Jour Livrable fil rouge
Jour 1 CLI Campus Quest : créer un joueur, accepter une quête, gagner de l'XP
Jour 2 API REST Campus Quest : endpoints joueurs/quetes/scores + classement
Jour 3 Atelier synthèse en binôme : mini-projet noté sur une après-midi

Idées de jeux Node.js (public 20 ans)

  • Campus Quest API : quêtes, XP, niveaux, leaderboard (idéal pour CLI puis API).
  • Meme Battle : vote de memes, score en temps réel, top hebdo.
  • Speed Trivia Duel : duel de quiz avec timer et points bonus.

Programme par jour

Jour 1 — Fondations Node.js

  • Outillage, modules, asynchrone

Jour 2 — API web avec Express

Jour 3 — Atelier synthèse en binôme

  • Expression des besoins
  • Mini planning de réalisation
