Node natif -> Express -> ateliers renforcés

Objectif pédagogique

Comprendre d'abord comment un serveur HTTP fonctionne sans framework, puis passer à Express pour industrialiser.

Exemple Node natif simple

import http from "node:http";

http.createServer((req, res) => {
  if (req.method === "GET" && req.url === "/health") {
    res.writeHead(200, { "content-type": "application/json" });
    return res.end(JSON.stringify({ status: "ok" }));
  }
  res.writeHead(404); res.end();
}).listen(3000);

Exemple Node natif technique

if (req.method === "POST" && req.url === "/players") {

On voit le coût de complexité du natif.

Passage à Express

import express from "express";
const app = express();
app.use(express.json());
app.get("/health", (_req, res) => res.json({ status: "ok" }));
app.listen(3000);
