Termes Docker à maîtriser

TermeDéfinition
ImageArtefact immutable en lecture seule.
ConteneurProcessus isolé lancé depuis une image.
LayerCouche créée par une instruction Dockerfile.
VolumeStockage persistant séparé du conteneur.
ServiceComposant défini dans compose.yml.

sandbox-node : démarrer et exécuter

# depuis la racine du projet
cd docs/nodejs-niveau-1/sandbox-node
docker compose up -d --build

docker compose exec node npm start -- help
docker compose exec node npm start -- list-quests
docker compose exec node npm start -- create-player Alice

sandbox-server : démarrer et tester

# depuis la racine du projet
cd docs/nodejs-niveau-1/sandbox-server
docker compose up -d --build

curl http://localhost:3000/
curl http://localhost:3000/health
curl http://localhost:3000/version

Arrêt et nettoyage

docker compose stop
docker compose down
docker compose down -v --rmi local --remove-orphans

docker system df
docker image prune -f
docker volume prune -f
docker builder prune -f

Les commandes de nettoyage suppriment des ressources : à utiliser avec prudence.
